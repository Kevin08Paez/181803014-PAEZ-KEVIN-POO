Puedes especificar imágenes de fondo
para cualquier elemento HTML visible.
En este ejemplo, la imagen de fondo
se especifica para un elemento div.
Por defecto, la imagen de fondo
se repetirá en la (s) dirección (es)
donde es más pequeño que el elemento
donde se especifica (Intente cambiar el tamaño del
ventana del navegador para ver cómo
La imagen de fondo se comporta.
